# hubot-gonews

Hubot golang news extension

See [`src/gonews.coffee`](src/gonews.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-gonews --save`

Then add **hubot-gonews** to your `external-scripts.json`:

```json
[
  "hubot-gonews"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
